## Usage

This project uses FastAPI, a lightweight and high-performance Python framework, for the backend.

### Install Dependencies

```bash
npm install
```

### Run Vite Frontend

React will run on http://localhost:3000

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
